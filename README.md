# 通学時刻表アプリ

## 概要
大学から最寄り駅までバスを使用する際、バスの時刻表をみてから、電車の到着時刻をブラウザで表示するのが手間に感じ、不便を解消するために開発したAndroidアプリケーションです。
ボタン1つで「現在時刻から乗車可能な直近のバス時刻と電車の時刻」を即座に表示します。

## こだわりポイント
* **リアルタイム時刻表照合:** 現在時刻とCSVデータの時刻表を照合し、最適な便を表示。

## 使用したもの
* **言語:** Kotlin
* **UI:** Jetpack Compose
* **データ:** CSV (Assets管理)
* **ツール:** Android Studio, Git

## 開発背景
複数ブラウザを開いて時刻を確認するのが面倒だと感じました。
「アプリ起動→即表示」というスピード感を重視し、自分専用のツールとして最適化しました。

## 起動方法
1. このリポジトリを `Clone` または `Download ZIP` します。
2. **Android Studio** を起動し、`Open` からプロジェクトフォルダを選択します。
3. エミュレータまたは実機（USBデバッグON）を選択して実行ボタンを押してください。
